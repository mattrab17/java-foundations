mkdir = make directory

git add . -> stage all changes
git commit -m "" -> commit changes to GitHub
git push -> push changes to GitHub repository
git status -> checks state of the repository

javac src/class.java -> compiles the java file into bytecode(.class)
java -cp src ArrayPractice -> launches the Java Virtual Machine to run the program 
                           -> -cp = class path
                           -> JVM executes your code starting from the main method
